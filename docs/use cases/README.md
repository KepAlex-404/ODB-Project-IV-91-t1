***НАЗВА:***  Створити обліковий запис.

***УЧАСНИКИ:***  Респондент, Система.

***ПЕРЕДУМОВИ:***  Обліковий запис відсутній.

***РЕЗУЛЬТАТ:***  Підтвердження облікового запису.

***ВИКЛЮЧНІ СИТУАЦІЇ:***
